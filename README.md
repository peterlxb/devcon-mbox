# devconnector

### Use React Mobx Node Express MongoDB

demo https://aqueous-mountain-96679.herokuapp.com/

## Teck stacks

- React 16.3
- NodoJS
- MongoDB
- Express
- Mobx

## Installation

- `git clone https://github.com/peterlxb/devcon-mobx`
- cd devcon-mobx
- npm install
- cd client
- npm install
- cd ..
- npm run dev
- visit `http://localhost:3000/`
